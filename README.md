# Watermelon Merge Game Prototype

This is a functional prototype of a Watermelon Merge game built in **Unity 6** for the Game Development Internship assignment.

## Features
* **Fruit Spawning:** Drop fruits from the top using mouse clicks.
* **Physics:** Realistic fruit behavior using 2D Physics Materials for bouncing and friction.
* **Merging Logic:** Identical fruits merge into a higher-level fruit upon collision.

## How to Play
1. Move the mouse to position the spawner.
2. **Left Click** to drop a fruit.
3. Combine identical fruits to evolve them!

## Technical Details
* **Engine:** Unity 6
* **Language:** C#
* **Assets:** Procedural Sprites (Circles and Squares)
